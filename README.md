# coding-practice

## ゴール

九九のように、考えなくても反射的に解けるレベルまで定着させることを目標とする。

## 学習手順

各問題に対して、以下の4ステップを3周する。

| 周 | やること | タイミング |
|---|---|---|
| 1周目 | 25分考える → 詰まったら解答を読む → 解法を要約してノート化 → **解答もノートも閉じて、何も見ずに最初から書き切る** | 初日 |
| 2周目 | 何も見ずに解き直す。15分超えたら 1周目のノートを見る | 翌日 |
| 3周目 | 何も見ずに 10分以内で解く。解けたら「定着」マーク | 7日後 |
| 維持 | 月1で再度解いて感覚を維持 | 30日後以降 |
